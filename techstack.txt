Tech stack:

Python 3.12
OpenCV — camera capture, frame processing, HUD rendering
MediaPipe — hand landmark detection (21 points per hand, up to 2 hands)
TensorFlow/Keras — EMNIST CNN model for air-writing character recognition
PyAutoGUI — keyboard/mouse automation (hotkeys, scroll, typing)
Win32 API (ctypes) — zero-latency cursor movement (SetCursorPos) and clicks (mouse_event)
scikit-learn — KNN classifier for personal gesture training
Streamlit — live dashboard UI
psutil + pygetwindow/win32gui — active app detection for context-aware profiles
pyttsx3 — offline text-to-speech voice feedback
multiprocessing Manager — shared state between main loop and dashboard
